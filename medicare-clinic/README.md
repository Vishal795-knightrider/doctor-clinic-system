# MediCare Clinic — Next.js Frontend

A modern SaaS-style doctor clinic website built with Next.js 14, Tailwind CSS, and React.

## Setup

```bash
cd medicare-clinic
npm install
npm run dev
```

Open http://localhost:3000

## Pages

| Route | Description |
|-------|-------------|
| `/` | Home page with Hero, Services, Testimonials, Timings |
| `/about` | Doctor profile and qualifications |
| `/services` | Detailed services listing |
| `/appointment` | Appointment booking form |
| `/contact` | Contact form and clinic info |
| `/login` | Admin login (demo: admin@medicare.com / Admin@123) |
| `/dashboard` | Admin dashboard overview |
| `/dashboard/appointments` | Appointments table with approve/cancel/delete |
| `/dashboard/patients` | Patient list with view/delete |
| `/dashboard/settings` | Profile and notification settings |

## Stack
- Next.js 14 (App Router)
- React 18
- Tailwind CSS
- Lucide React icons
- DM Serif Display + DM Sans fonts
